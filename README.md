# Secure Network Infrastructure & Security Services 🛡️🌐

## Overview
This repository contains the documentation, logbook, and final report for the **Workshop II (BITU 3923)** project developed at Universiti Teknikal Malaysia Melaka (UTeM)[cite: 4]. The project focuses on the fundamentals of designing, implementing, and securing network systems to enhance data privacy and system stability[cite: 4]. 

## Key Features & Implementations
* **Network Architecture:** Designed a scalable network utilizing multiple VLANs including Wireless, IT Staff, Server, and Guest networks[cite: 4].
* **Servers & Operating Systems:** Configured Windows Server 2016, Ubuntu 22.04 LTS, and Kali Linux 2024.4 to host and manage 20 distinct network services[cite: 4].
* **Security Services:** Implemented Web Application Firewall (WAF), RADIUS server for robust authentication, and Single Sign-On (SSO) integrated with Active Directory[cite: 4].
* **Intrusion Detection System (IDS):** Deployed and configured Suricata, integrating it seamlessly with the Elastic Stack (Elasticsearch, Kibana, Filebeat) for real-time traffic monitoring and incident response[cite: 4, 5].
* **Network Hardening:** Applied strict security policies including Access Control Lists (ACL), NAT, IPSec VPN configurations, Port Security, and system hardening for both Windows and Linux environments[cite: 4].

## Project Documentation
* `Final Report`: Details the logical and physical network design, hardware/software specifications, security policies, and step-by-step service configurations[cite: 4].
* `Logbook`: Chronicles the weekly progress, troubleshooting, and testing phases of the project, specifically detailing the IDS and SIEM dashboard setup[cite: 5].
* `Certificate`: Includes the Certificate of Participation in The Workshop II Innovation Competition awarded by the Faculty of Information and Communication Technology[cite: 6].
